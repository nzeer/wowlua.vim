WTF is this?
============

Lua syntax file with [World of Warcraft](http://www.worldofwarcraft.com/) apis for Vim editor

How to use?
===========

Download [this file](http://github.com/yaroot/wowlua.vim/raw/master/lua.vim) and save it to *$VIMRUNTIME/syntax/* .
